# **App Name**: Dormammu Dashboard

## Core Features:

- kWh Gauge: Display a gauge representing kilowatt-hour (kWh) usage retrieved from the Dormammu Firebase database.
- PHP Consumption Gauge: Display a gauge representing consumption in PHP currency, converted from the kWh value in Firebase.
- Firebase Connection: Connect to the Dormammu Firebase realtime database to fetch kWh data.
- Currency Conversion: Implement logic to convert kWh usage to PHP currency based on a predefined or dynamically fetched exchange rate.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey trust, reliability, and technological sophistication.
- Background color: Very light grayish-blue (#E6E8EA), to provide a subtle and clean backdrop that allows the vibrant primary and accent colors to stand out.
- Accent color: Vibrant Violet (#8A2BE2) used for highlights, interactive elements, and key data points.
- Body and headline font: 'Inter', a grotesque-style sans-serif font to provide a clean and modern feel that is suitable for both headlines and body text.
- Use clear and simple icons to represent different data points and functionalities.
- Design a clean and intuitive layout that presents data in a clear and understandable format.
- Use subtle transitions and animations to enhance the user experience and provide visual feedback.
