# **App Name**: DeviceBase

## Core Features:

- Device List: Display a list of devices with key attributes such as serial number and device type.
- Device Details: View detailed information for each device.
- Add Device: Add new device records by inputting serial number and device type, as well as other attributes.
- Edit Device: Modify existing device records, updating the attributes of each device.
- Delete Device: Remove device records.

## Style Guidelines:

- Primary color: Subtle Blue (#778DA9) to invoke a sense of reliability and trustworthiness.
- Background color: Light Gray (#E0E1DD) for a clean, modern, professional look.
- Accent color: Soft Lavender (#995FBC) to highlight key actions without overwhelming the UI.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used for its modern and neutral appearance.
- Use simple, outline-style icons for device actions.
- Implement a clear, card-based layout to display device information. Maximize whitespace and legibility.
